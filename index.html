Final Project

Final project includes Bootstrap concepts, Google Fonts, along with Map rendered using CartoDB dataset. Toggle button is provided to switch between layers where layer-1 displays the NYC Boroughs and layer-2 displays the Restaurants by grade. Upon hover over the map points, info window will display additional details.
